# ComposeCA2

A simple Android registration form app built with Jetpack Compose and Material Design 3.

## Features

- Registration form with name, email, and phone number fields
- User type selection (Student/Professional)
- Form validation with error messages
- Success notifications via Snackbar

## Requirements

- Android Studio
- Minimum SDK: API 24 (Android 7.0)
- Target SDK: API 36 (Android 14)
- JDK 11+

## Getting Started

1. Open the project in Android Studio
2. Sync Gradle files
3. Run the app on an emulator or device

## Usage

Fill in all fields (name, email, phone) and select a user type, then click "Register". The app will validate the form and show a success or error message.

## Tech Stack

- Kotlin
- Jetpack Compose
- Material Design 3
